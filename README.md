# N-Body-Solver

Celestial trajectory solver aiming for gpu optimization

# Commande pour lancer le projet pour la première fois

mkdir build
cd build
cmake ..
make
./nbody_solver

# Pour les fois suivantes

rentrer dans le dossier build
taper : make

# Si tu creer un nouveau fichier ou change les shader

rentrrer dans le dossier build
taper : cmake ..
make
